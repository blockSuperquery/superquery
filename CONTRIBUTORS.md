# Contributors

This project is derived from the SubQuery Stellar indexer (`subql-stellar`).
On 2026-07-03 the git history was truncated to a clean 2024-onward baseline for
this repository's fresh start. Truncating history necessarily drops older commits
from the commit log, so this file preserves credit for **everyone** who
contributed across the complete original history (1,839 commits, 2020–2026).

Commit counts are from the full pre-truncation history of the `main` branch.
Multiple git identities belonging to the same person have been consolidated.
